## Legend API


### attr 属性

| 名称     | 说明                             | 参数                                             |
| -------- | -------------------------------- | ------------------------------------------------ |
| pos | `Legend` 位置控制                | `(type: 'left' | 'top', value: String | Number)` |
| vertical | 控制 `Legend` 水平布局或垂直布局 | `(flag: Boolean)`                                |


#### style 属性：
| 名称         | 说明                 | 类型  |
| ------------| --------------------| ----- |
| css-icon       | 图例    | 由Circle绘制       |
| css-icon:hover      | 图例hover |      |
| css-text      | 文本       | 由Label绘制      |
| css-text:hover      | 文本hover       |       |
